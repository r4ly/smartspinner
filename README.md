# Smart Spinner

Smart Spinner is a stupid ideas hackathon app that pretends to make smart decisions while always nudging you toward chaos.

It ships with two modes:

- Gravity Mode: the wheel acts like "heavy" bad ideas sink to the bottom.
- AI Worst Choice: fake agent logic that confidently picks the worst option.

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Demo Flow

1. Select a mode.
2. Click "Spin Me Into Regret".
3. Read your terrible life advice with maximum confidence.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS + custom CSS animation
